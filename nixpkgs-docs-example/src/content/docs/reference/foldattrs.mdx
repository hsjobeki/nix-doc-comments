Apply fold functions to values grouped by key.

# Example

```nix
foldAttrs (item: acc: [item] ++ acc) [] [{ a = 2; } { a = 3; }]
=> { a = [ 2 3 ]; }
```

# Type

```haskell
foldAttrs :: (Any -> Any -> Any) -> Any -> [AttrSets] -> Any
```

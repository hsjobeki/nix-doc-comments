Like `traceSeq`, but only evaluate down to depth n.
This is very useful because lots of `traceSeq` usages
lead to an infinite recursion.

# Example

```nix
traceSeqN 2 { a.b.c = 3; } null
trace: { a = { b = {â€¦}; }; }
=> null
```

# Type

```
traceSeqN :: Int -> a -> b -> b
```

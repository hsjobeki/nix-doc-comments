Call a function for each attribute in the given set and return
the result in a list.

# Example

mapAttrsToList (name: value: name + value)
{ x = "a"; y = "b"; }
=> [ "xa" "yb" ]

# Type

mapAttrsToList :: (String -> a -> b) -> AttrSet -> [b]

Utility function that creates a `{name, value}` pair as expected by `builtins.listToAttrs`.

# Example

nameValuePair "some" 6
=> { name = "some"; value = 6; }

# Type

nameValuePair :: String -> Any -> { name :: String; value :: Any; }

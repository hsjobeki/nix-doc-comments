Check whether the argument is a derivation. Any set with
`{ type = "derivation"; }` counts as a derivation.

# Example

```nix
nixpkgs = import <nixpkgs> {}
isDerivation nixpkgs.ruby
=> true
```

```nix
isDerivation "foobar"
=> false
```

# Type

```haskell
isDerivation :: Any -> Bool
```

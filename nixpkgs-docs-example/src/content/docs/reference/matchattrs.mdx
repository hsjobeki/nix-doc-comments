Returns true if the pattern is contained in the set. False otherwise.

# Example

```nix
matchAttrs { cpu = {}; } { cpu = { bits = 64; }; }
=> true
```

# Type

```haskell
matchAttrs :: AttrSet -> AttrSet -> Bool
```

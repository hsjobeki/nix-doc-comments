Generate an attribute set by mapping a function over a list of
attribute names.

# Example

genAttrs [ "foo" "bar" ] (name: "x_" + name)
=> { foo = "x_foo"; bar = "x_bar"; }

# Type

genAttrs :: [ String ] -> (String -> Any) -> AttrSet

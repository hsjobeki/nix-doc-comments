If `cond` is true, return the attribute set `as`,
otherwise an empty attribute set.

# Example

```nix
optionalAttrs (true) { my = "set"; }
=> { my = "set"; }
optionalAttrs (false) { my = "set"; }
=> { }
```

# Type

```haskell
optionalAttrs :: Bool -> AttrSet -> AttrSet
```
